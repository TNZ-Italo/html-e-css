ADICIONANDO LINKS INTERNOS

acesse agora minha segunda pagina clicando aqui!

para pode criar links internos, siga os passos

  1. crie uma nova pagina de html, dentro do mesmo arquivo
  2. volta a pagina 1
  3. puxe a tag ~a~
  4. dentro de href, ctrl + espaço
  5. seleciona a pagina
  6. e atenção para nomear no local correto.
  7. na outra pagina fazer a mesma coisa

nessa situação, não se usar a tag target_blank e rel=external por ser links internos

vc devera seguir os proximos passos:

  1. depois do link, antes do simbolo >nome em azul<
  2. vc escrever rel="next" na primeira pagina
  3. e escrever rel="prev" na segunda pagina

isso serve para orintar melhor o navegador

tambem existe um nofollow, que se colocar depois do rel, que significa que vc não da um aval sobre o conteudo

é importante que em uma situação que link, esteja dentro de uma outra pagina na area de trabalho, se deve colocar todo o caminho corretamente

tambem é possivel que para o usuario não carregue varias paginas em seu navegagor, basta colocar dessa maneira, target´_self´ que significa na propria pagina

outra dica, quando vc estiver dentro de uma pagina que foi aberta separada das comuns, para pode volta as pagina anterior é importante que dentro de href vc coloque ../ para pode aparecer as outras paginas
