adicionando audio...

para adicionar audios, é importante ter um cuidado com os direitos, assim como vale para imagens:

  • precisa ter em mente que nem todo navegador tem a compatibilidade
  • para ter uma compatibilidade maior, é necessario sabe mudar os formatos dos audios para MP3, WAV,OGG
  • usando a propria tag audio

dentro de audio ainda é possivel adicionar alguns comandos

  • preload, dentro dele tem os seguintes comandos
  • auto , so vai considerar que o audio carregou, depois de carregar todo o site
  • metadata , so ira carregar algumas inf como data, duração
  • none , não ira carregar nada, apenas quando o usuario apertar play
  • loop , para o audio fica recomeçando ao chegar no final
  • NAO USA WAV POR CONTA DO ENORME TAMANHO
